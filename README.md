# homebrew-tap

Homebrew tap for ilyakooo0 packages.

## Usage

```
brew tap ilyakooo0/tap
```

### Parseable Viewer

A macOS client for [Parseable](https://github.com/ilyakooo0/parseable-macos).

```
brew install --cask parseable-viewer
```

### SourceDocsSwift

```
brew install sourcedocsswift
```
